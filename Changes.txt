Changelog for Version 20.0 (Major)

Additions/changes:
Updated to TPT 96.0 (enabling uploading saves with CGoL elements.)
New option: fancy cursor (Draws fancy graphics around pointer, changes colour as per theme.)
New elements: CSNS, ctype sensor (Also worth mentioning all these new sensor support serialisation.)
Updated texter (now supports typing multiple lines of text.)
Themes now change in realtime, even when menu is kept open (helpful when using custom settings.)
Now you can click outside the menu to close it.
Seeds grow more naturally.
Add support for meson.
Scripts now use lua event api functions instead of older tpt.register ones.
Moved "MORT" into gas menu.
Clouds now creates LIGH in a more realistic way.
Performance Improvements.
New intro design.

Fixes:
Fix out of range errors when typing really long sentences in texter.
Various under the hood changes to script.
Add a lowerlimit for brightness settings.
Fix topbar not hiding even when turned off while using pulse theme.
